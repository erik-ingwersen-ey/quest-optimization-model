Datatools
=========

Modules used for **input data ingestion**, **input data validation**, **data cleanning** and **creating all necessary columns**.
Methods created for data manipulation are stored at :meth:`optimization.datatools.dataprep <optimization.datatools.dataprep>`.
Using the methods from :meth:`optimization.datatools.dataprep <optimization.datatools.dataprep>`, :meth:`optimization.datatools.pipelines <optimization.datatools.pipelines>` stores
all pipelines used to load and manipulate input inventory reports that are then passed to the optimization model.
